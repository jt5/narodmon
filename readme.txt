Библиотека для отправки данных с шилда «Cosmo GSM Connect» на онлайн сервис «Народный мониторинг».
https://github.com/jt5/narodmon

Архив «narodmon» содержит библиотеку «GSM» и ряд файлов с процедурами обработки данных для сервиса «Народный мониторинг» http://narodmon.ru/

Установка библиотеки в Arduino IDE:
1) Скачайте из репозитория https://github.com/jt5/narodmon архив с программой.
2) Разархивируйте и переименуйте директорию «narodmon» в «GSM».
3) Откройте директорию, в которой установлена Arduino IDE. Найдите папку «libraries» и скопируйте в нее архив «GSM».
4) Запустите среду программирования Arduino IDE. Перейдите на вкладку «Файл» -> «Примеры» -> «GSM» и выберите, например, скетч «NarodmonTemperature».

Подробная инструкция по работе с библиотекой - https://github.com/jt5/narodmon
