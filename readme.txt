GSM library for Arduino.

Библиотека под Ардуино и шилд Cosmo GSM Connect.

Features:
- Регистрация в GSM-сети
- Отправка SMS
- Прием и разбор SMS
- Прием входящих вызовов
- Отправка данных по GPRS.
- Проверка статуса соединения с сетью GSM, восстановление сети при потере сигнала
- Выполнение произвольных AT-команд из скетча

Required:
- Плата Arduino (Uno, Duemilanove, Cosmo Black Star и другие совместимые аналоги с Atmega328).
- Шилд "Cosmo GSM Connect".
